Primeiramente, é preciso retirar do arquivo de entrada .csv o(s) atributo(s) classe, caso exista(m).
Para então executar o programa, deve-se executar pelo terminal o comando:

    python3 src/runme.py < arquivo_entrada > arquivo_saida

onde:
    arquivo_entrada é o arquivo .csv de entrada,
    arquivo_saida é o nome do arquivo de saída txt com o agrupamento feito.

Exemplo do comando no terminal: 

    python3 src/runme.py < example.csv > out.txt
